Тестовое задание на позицию разработчика в тестировании (Программист-тестировщик).  
Выполнил Никулин Дмитрий Валерьевич

Структура проекта
downloads - папка для скачивания файлов 
logs - папка для хранения log-файлов (loguru)
pages - папка для хранения страниц POM
reports - папка для хранения данных для генерации отчета (allure)
tests - папка для тестов (pytest/selenium)
conftes.py - конфигурация тестов драйвер определяется сам (webdriver_manager) подготовлен для Chrome(--headless)
